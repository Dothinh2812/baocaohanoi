════════════════════════════════════════════════════════════════════════════════════
                              API CALLS SEQUENCE
════════════════════════════════════════════════════════════════════════════════════

1. Telegram Bot → Application (webhook POST)
   POST /telegram-webhook

2. Application → Telegram (get file)
   https://api.telegram.org/file/bot{TOKEN}/{file_path}

3. Application → Google Cloud Storage (upload)
   bucket.blob().upload_from_string()

4. Application → Google Vision API (OCR)
   vision_client.text_detection(image)

5. Application → OpenAI API (analysis)
   client.chat.completions.create()

6. Application → Google Sheets API (save)
   sheet.append_row()

7. Application → Telegram Bot API (send message)
   POST https://api.telegram.org/bot{TOKEN}/sendMessage

8. Application → N8N Webhook (optional notification)
   requests.get(WEBHOOK_URL)
